# 🎬 AluraPlus

Projeto desenvolvido durante os estudos de Front-End, criando uma página de divulgação da plataforma Alura+.

## 🚀 Acesse o projeto

🔗 [Clique aqui para visualizar o site](https://talissonengelmann.github.io/aluraplus/)

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## 📚 Sobre o projeto

O AluraPlus é uma landing page responsiva criada para apresentar os planos da plataforma Alura+, com foco em prática de estruturação HTML, estilização CSS e organização de layout.

Durante o desenvolvimento foram aplicados conceitos como:

- Estrutura semântica HTML
- Flexbox
- Grid Layout
- Responsividade
- Organização de arquivos CSS
- Boas práticas de desenvolvimento Front-End

## 📱 Responsividade

O projeto foi desenvolvido para se adaptar a diferentes tamanhos de tela:

- Desktop
- Tablet
- Mobile

## 🎨 Interface

A página apresenta:

- Seção principal de apresentação
- Divulgação dos planos
- Sessões explicativas com imagens
- Área de dispositivos compatíveis
- Rodapé com links
